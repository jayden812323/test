# 標題關鍵字搜尋工具

針對 PTT 脆熱議版的文章標題，進行多條件關鍵字篩選的命令列工具。

## 功能

- 支援 OR、AND、NOT 邏輯組合搜尋
- 支援括號群組
- 支援近似搜尋（兩詞之間允許間隔 N 個字）

## 搜尋語法

| 符號 | 意義 | 範例 |
|------|------|------|
| `+` | OR（符合其中一個） | `美食+旅遊` |
| `&` | AND（同時符合） | `美食&台北` |
| `-` | NOT（排除） | `美食-素食` |
| `(a;b)` | 括號內用 `;` 做 OR | `(美食;小吃)&台北` |
| `a/N/b` | 近似搜尋，兩詞間隔最多 N 字 | `美食/3/推薦` |

## 使用方式

### 環境需求

- Python 3.8+
- pandas

安裝套件：

```bash
pip install pandas pyarrow
```

### 準備資料

將資料檔 `脆熱議all.parquet.gzip` 放在與程式相同的資料夾，並修改 `完整.py` 第 4 行的路徑：

```python
df = pd.read_parquet("脆熱議all.parquet.gzip")
```

### 執行

```bash
python 完整.py
```

出現 `>>>` 後輸入搜尋條件，按 Enter 顯示結果。

## 搜尋範例

```
>>> 美食
>>> 美食+旅遊
>>> 美食&台北-素食
>>> (美食;小吃)&台北
>>> 美食/3/推薦
```

## 輸出說明

顯示符合條件的文章標題與留言數，最多顯示前 10 筆。

```
篩選結果：42 筆
title                          comment_count
...
```
